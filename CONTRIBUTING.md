# Contributing to CornVision AI

First off, thanks for taking the time to contribute! We welcome all contributions that help improve industrial quality control.

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template in GitHub Issues.
- Describe the environment (Windows/Linux/Mac) and the steps to reproduce.

### Suggesting Enhancements
- Open an **Enhancement Suggestion** in Issues.
- Explain why this feature would be useful for cornflake manufacturing or AI quality control.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the PEP 8 (Python) and ESLint (JS) standards.
3. Update the README.md if you are changing any core functionality.
4. Link your PR to the original issue.

## Development Setup
Check the [README](README.md#manual-build) for instructions on setting up your local development environment.
